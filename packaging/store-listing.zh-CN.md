## 简短描述

为 Microsoft Store ChatGPT 桌面应用提供独立的进程级代理启动，不修改 Windows 系统代理。

## 描述

GProxyT 是一款轻量的 Windows 桌面启动工具，为 Microsoft Store ChatGPT 桌面应用及其子进程单独配置 HTTP 代理。

它不会修改 Windows 系统代理，也不会更改其他应用的网络设置。你可以保存本地代理地址，并通过一个简洁的启动按钮重新打开 ChatGPT。

主要功能：

- 仅为 ChatGPT 进程注入代理配置
- 不修改系统代理和用户环境变量
- 自动识别当前用户安装的 Microsoft Store ChatGPT
- 支持自定义 HTTP 代理地址
- 支持登录 Windows 后自动通过代理启动
- 代理可用性检查与清晰的错误提示
- 配置仅保存在本机

使用前需要安装 Microsoft Store ChatGPT，并准备可访问的本地或远程 HTTP 代理。

GProxyT 是独立的第三方工具，与 OpenAI 不存在隶属、赞助或认可关系。ChatGPT 和相关标志归其各自权利人所有。

## 功能

- ChatGPT 进程级代理
- 不修改 Windows 系统代理
- 自动定位 Microsoft Store 应用
- 自定义代理地址
- 可选开机启动
- 本地配置存储

## 系统要求

- Windows 10 版本 2004 或更高版本
- Windows 11
- x64 处理器
- Microsoft Store ChatGPT 桌面应用
- HTTP 代理服务

## 隐私

GProxyT 不要求创建账户，不收集、上传、出售或共享个人数据。代理设置保存在当前 Windows 用户的本地应用数据目录。只有用户主动启用调试模式时才会在本机生成诊断日志，日志不会由 GProxyT 自动上传。
